Get Started
Runing jsinspector server yourself
Installing
git clone https://github.com/switer/jsinspector.git jsinspector
Starting server
JSInspector server require node.js. After install node , runing server with npm (node.js package manager).
cd ./jsinspector
npm i
npm start
Visit Home Page and Inject the script
Inject the script (which has an inspector id) from home page to remote website, then visit (your can get then link on home page):
https://yourhost:port/devtools?xxx-xxx
